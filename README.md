本專案為「演算法」課程的作業 4 與作業 5 進度報告。
[cite_start]內容展示了如何透過系統化的 IEEE 文獻回顧（作業 5）作為唯一羅盤，直接引導並收斂出期末專題的初步研究架構（作業 4） [cite: 2, 5, 7, 9][cite_start]。本專案針對機器學習與邊緣運算領域進行探討，並提出動態自適應邊緣運算框架的初步構想 [cite: 12, 31]。

## 🧠 核心研究與技術矩陣 (Core Research & Tech Matrix)
[cite_start]本專案透過 IEEE Xplore 收斂出關鍵文獻，並進行了以下技術矩陣對比與缺口分析 [cite: 11, 19]：

* [cite_start]**文獻 A (Chen et al., 2022):** 採用靜態神經網絡壓縮（提升 15% 推理速度），但受限於運算成本過高 [cite: 20]。
* [cite_start]**文獻 B (Wang & Li, 2023):** 採用雲端聯合學習架構（92% 準確率），但缺乏即時數據處理能力 [cite: 20]。
* [cite_start]**文獻 C (Kumar et al., 2023):** 採用基於規則的資源分配（降低 10% 延遲），但應用場景受限於硬體 [cite: 20]。
